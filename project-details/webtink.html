A Shop Front Website

I made this website on WordPress platform, as a shop front website when I was working as a freelancer. I provided WordPress and OpenCart websites development, domain registration and hosting services.

Fluid Layout

The layout of this website is fluid, the width of the container uses relative unit size (percentage).

Subtle Effects

I used subtle effects, such as sliding menu, ease-in-out button effect, etc. to avoid unwanted distractions yet adding a little bit of dimension on user experience perspective.

Choice of Colour

The main colour used in the website is green (HEX code #16C0A3), the accent is black (HEX code #111111) and grey (HEX code #3b3b3b).
