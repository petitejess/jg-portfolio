A Demonstration of CSS and JavaScript Usage

This project was made to demonstrate the capability of CSS in creating shapes and simple animations, as well as the capability of Javascript to create a browser game.

CSS Shapes

All shapes in this website were made using CSS. Most of them were made by manipulating the CSS properties/pseudo elements:

  • ::before, ::after
  • position
  • box-shadow
  • border-radius
  • element size
  • transform
  • nested elements

CSS Animation

Animations done on the shapes were made using CSS animation keyframes, utilising functions such as:

  • scale
  • rotate
  • translate

JavaScript 2D Browser Game

I used Phaser library to create a prototype platform browser game. Currently, it can only be displayed on desktop or screen larger than 1200px wide.
